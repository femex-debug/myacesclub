# ACES Club — Live Leaderboard

Real-time tennis leaderboard for ACES Club Inc., Laurel MD.
Powered by Firebase Firestore.

## Deployment
1. Push this folder to a GitHub repository
2. Connect the repo to Netlify
3. Deploy — no build step needed, it is plain HTML/JS

## First Time Setup
1. Open the live site
2. Click the 🔒 icon in the top right
3. Click "First time? Set admin PIN"
4. Choose a PIN (min 4 digits) — store it safely
5. You are now in admin mode

## Admin Features
- Add players (by username)
- Schedule matches
- Enter match scores
- Create tournament brackets
